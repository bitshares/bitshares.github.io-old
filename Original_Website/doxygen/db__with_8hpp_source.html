26  * This file provides with() functions which modify the database
27  * temporarily, then restore it. These functions are mostly internal
28  * implementation detail of the database.
29  *
30  * Essentially, we want to be able to use "finally" to restore the
31  * database regardless of whether an exception is thrown or not, but there
